## SetAside
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

# Motivation

Nearly 12 million Americans make their primary income as 1099 workers, or independent contractors. One of the biggest drawbacks of working as a 1099/independent contractor is the fact that taxes are not taken out of the worker's paycheck, requiring the worker to calculate estimated taxes to be sent in quarterly. SetAside is a tool intended to help with estimating the amount of self-employment taxes to be sent by the 1099 worker or independent contractor. 

# Quick Start

Currently, SetAside is a command-line tool which calculates the amount that an individual should set aside for self-employment taxes. To use this tool, ensure that you have Python installed on your system and run freelance-tax-estimator.py. Future iterations will focus on adding UX through use of an interface.

# Usage

Upon running freelance-tax-estimator.py, the user simply enters his or her weekly, monthly, quarterly, or annual earnings, and receives a number back estimating how much of that income should be set aside for tax purposes.
